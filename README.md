# YuemingHub 产出索引

这是一个纯静态作品索引站，用来串起三个主要入口：

1. `向未来家庭教育`：品牌官网，负责信任与咨询转化
2. `Gui`：个人私人工具站，负责回到自己、清噪、收束节奏
3. `爱与规则四层成长系统` 与家庭教育样本研究：长期沉淀的方法与边界

## 技术栈

- 纯静态 HTML
- CSS：`css/style.css`、`css/cosmos.css`
- 原生 JavaScript：`js/site-shell.js`、`js/site-ui.js`、`js/main.js`

## 页面结构

- `index.html` — 产出索引首页
- `README.md` — 站点说明
- `css/` — 视觉系统
- `js/` — 头尾注入与交互

## 站点职责

- 这里不是品牌官网，也不是个人工具站本体
- 这里负责把所有作品串起来，让访问者知道该从哪里进入
- 公开内容不直接暴露样本库隐私，只展示研究方向与判断边界
